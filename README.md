# 2023MORAIContest

### Requirements
```
rosdep install --from-paths . --ignore-src -r -y
```

### Installation
```
$ git clone git@github.com:lovelyoverflow/2023MORAIContest.git
$ cd ./2023MORAIContest
$ catkin_make
```

### Run
```
$ source ./devel/setup.zsh
or
$ source ./devel/setup.bash

$ rosrun main main.launch
```

### 빌드 오류 시
devel 폴더를 지우고 재빌드
